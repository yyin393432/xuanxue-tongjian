# 玄学通鉴 · Chinese Metaphysics Toolkit

一个**纯前端、离线、开源**的中国传统命理工具集。覆盖八字、紫微、六爻、奇门、易经、称骨、五行、神煞、生肖、星座、姓名、生日、风水、解梦、测字、数字能量、合婚、流年、面相、前世今生等方向。

> ⚠️ **免责声明**：本工具仅供**娱乐与文化研究**之用。所有结果均由传统命理规则与公开算法自动推算，**不代表、不提供任何医疗、财务、法律或人生建议**。命理玄学尚无科学依据，请理性看待，切勿迷信，更勿据此做出重大人生决策。

## 特性

- **完全离线**：所有计算在浏览器本地完成，无需联网、无需后端、无数据上传。
- **历法准确**：月柱以**节气**为界、年柱以**立春**为界（非春节），使用 `lunar-javascript` 离线历法库。
- **问答制输入**：顶部「个人档案」只需填一次（公历生日 / 时辰 / 性别 / 出生地 / 姓名），切换模块自动保留，无需重复填写；记不清出生时辰可选「不详」，系统自动标注影响范围。
- **真太阳时校正**：填写出生地（内置主要城市经度表）后，八字按「真太阳时」排盘——经度偏西会让时辰提前，可能跨时辰甚至跨日，已在报告中标注「北京时 → 真太阳时」校正过程。
- **神煞计数**：神煞不仅判命中，还统计命带几处（如「天乙贵人 ×2」），并列出落在哪一柱、查法来源。现行 **14 种神煞**：
  - 年支/日支查（三合局）：桃花(咸池)、驿马、华盖、将星、劫煞、灾煞
  - 年干/日干查：天乙贵人、文昌
  - 日干查（帝旺位）：羊刃
  - 年支查：孤辰、寡宿
  - 月支查（干支混合）：天德贵人、月德贵人
  - 日柱查：空亡(旬空)
  - 查法口径均透明展示在报告中（年支/日支、年干/日干、月支），便于与权威资料核对。
- **先科普后结果**：每个模块先讲原理，再给个性化结果，拒绝"黑箱算命"。
- **白话文讲解**：每个模块的结果下方都附「白话文讲解（通俗解读）」折叠卡，先用大白话讲清原理，再结合你的真实结果做通俗解读，普通人也能看懂。
- **架构解耦**：纯计算核心与界面分离，便于将来移植到 APP / EXE / 小程序 / 收费版。
- **开源可扩展**：Apache-2.0，模块按统一模板接入，欢迎社区贡献。

## 用法

直接用浏览器打开 `index.html` 即可，**无需安装、无需服务器**。

```bash
# 任选一种：双击 index.html，或本地起个静态服务
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 计算核心的可移植性

`js/core/*` 与 `js/data/*` 不依赖任何 DOM / 浏览器 API，是纯函数模块。因此同一套算法可以：

- 直接用于 **Electron / Tauri 打包成 EXE / dmg**；
- 通过 **Capacitor / React Native** 封装成手机 APP；
- 在服务端（Node.js）复用做 **API / 收费版**；
- 接入 WorkBuddy 等平台作为 **SKILL**。

只需把对应语言/运行时的历法库替换为等价实现（如 Python 的 `lunar_python`），核心算法逻辑一致。

## 贡献指南（如何加一个新模块）

1. 在 `js/core/` 下新建 `xxx.js`，用与 `bazi.js` 相同的 UMD 包装，导出 `compute(input)` 纯函数。
2. 在 `js/data/` 下放批语 / 文案数据（如需）。
3. 在 `js/ui/app.js` 的 `MODULES` 数组登记模块（填 `id` / `name` / `desc` / `status`）。
4. 在 `selectModule()` 中为该 `id` 渲染表单并调用 `compute` + `renderReport`。
5. 提交 PR，请附上计算结果的样例以便校对。
